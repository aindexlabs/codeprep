# **App Name**: CodePrep

## Core Features:

- Curated Question Sets: A collection of interview questions covering JavaScript, TypeScript, React, React Native, and infrastructure, ranging from basic to advanced levels.
  - JavaScript: https://developer.mozilla.org/en-US/docs/Web/JavaScript
  - TypeScript: https://www.typescriptlang.org/docs/
- Example Code Snippets: Provide example code snippets to illustrate solutions and best practices for each question.
- Personalized Learning Paths: Adaptive learning paths based on user skill level and preferred technology stack to learn. Uses tool that intelligently suggests relevant practice questions based on performance.
- Practice Environment: A sandboxed code editor or environment with syntax highlighting where users can write and test code in real-time, ensuring immediate feedback.

## Style Guidelines:

- Primary color: Dark blue (#243A73), conveying trustworthiness and intelligence.
- Background color: Very light blue (#E5E8EF), almost white.
- Accent color: Bright cyan (#00D1FF), emphasizing key actions.
- Body: 'Inter' (sans-serif), to ensure clean readability.
- Headlines: 'Space Grotesk' (sans-serif), for a tech-forward aesthetic. Should Space Grotesk become illegible at small sizes, use Inter (sans-serif) as fallback.
- Code snippets: 'Source Code Pro' (monospace), for better distinction from UI text
- Consistent usage of system-standard icons, emphasizing simplicity and familiarity.
- Subtle transitions and animations for feedback on user interactions, like submitting or checking answers.
